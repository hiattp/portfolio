My Skill Set

FULL STACK DEVELOPMENT

Ruby on Rails

Server Framework

Rails is a popular server-side framework written in Ruby. Over the past few years it has become a robust, scalable solution for many common web paradigms.

HTML(5)

Frontend Markup

This is the raw content of your web pages (the words, images, etc). HTML has become commoditized with the rise of outsourced development, but good markup still pays dividends.

CSS(3)

Frontend Styling

CSS controls how your pages look (think color and position). Want to see what a website looks like without styling? Click here to see this page without CSS.

JavaScript

Interaction

JavaScript is a jack of all trades. It does cool animation and UX enhancement on the frontend, and occassionally powers neat server-side features like real-time messaging.

Node.js

Server Framework

Node is a flexible server-side framework written in JavaScript. While not appropriate for all scenarios, Node can be an elegant solution for use cases like lightweight JSON APIs and "soft" real-time games.

AngularJS

Dynamic Markup

HTML was not originally designed for the interactive, dynamic web applications we use today. AngularJS extends HTML to facilitate the smooth, seamless UX that contemporary web users expect.

jQuery

DOM Manipulation

Since HTML is not interactive by nature, we can use JavaScript to add cool interactivity without fully extending it with something like AngularJS. jQuery is a powerful JS library of tools that do just that.

PostgreSQL

SQL Database

Postgres is a powerful, battle-tested database technology that drives many large-scale implementations. This is my go-to for a robust relational database.

MongoDB

NoSQL Database

Mongo is a leading document-oriented database, and provides a robust solution for semi-structured data storage. I like the way it interfaces with Node.js, since the entire stack can be written in JS.

Firebase

Real-time DaaS

Firebase provides a "database as a service" technology with real-time interaction in mind, making it easy to get up and running with real-time features using pure client-side JS.

Chrome

Browser Extensions

Browser extensions can be amazingly powerful tools if used and built correctly. They provide a unique venue both to enhance your user experience and to increase your brand presence.

PhoneGap

Mobile Apps

Native mobile development is challenging and expensive because each phone OS requires a different language and SDK. I use PhoneGap to build cross-platform mobile apps that look and feel native.
